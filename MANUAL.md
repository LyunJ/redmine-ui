# Redmine UI - 사용 매뉴얼

Redmine의 사용성 개선을 위한 크로스 플랫폼 데스크톱 클라이언트입니다.
System tray에 상주하며, 글로벌 단축키로 빠르게 호출할 수 있는 경량 앱입니다.

---

## 1. 설치 및 실행

### 설치

빌드된 설치 파일 중 하나를 사용합니다.

- **MSI**: `Redmine UI_x.x.x_x64_en-US.msi` (Windows Installer)
- **EXE**: `Redmine UI_x.x.x_x64-setup.exe` (NSIS Installer)

### 실행

설치 후 앱을 실행하면 System tray(작업 표시줄 알림 영역)에 아이콘이 나타납니다.

---

## 2. 로그인

앱 최초 실행 시 로그인 화면이 표시됩니다.

| 항목             | 설명                                                  |
| ---------------- | ----------------------------------------------------- |
| **Redmine URL**  | Redmine 서버 주소 (예: `https://redmine.example.com`) |
| **API Key**      | Redmine 개인 API Key                                  |

### API Key 확인 방법

1. Redmine 웹 접속 > 우측 상단 **내 계정** 클릭
2. 우측 패널 **API 접근키** > **표시** 클릭
3. 표시된 키를 복사하여 입력

> 로그인 정보는 앱 내부에 안전하게 저장되며, 다음 실행 시 자동 로그인됩니다.

---

## 3. 메인 화면 (일감 목록)

로그인 후 상단에 **뷰 탭**이 표시되며, 탭을 선택하여 다양한 일감 목록을 확인할 수 있습니다.

### 뷰 탭

| 탭                    | 설명                                         |
| --------------------- | -------------------------------------------- |
| **해야할 일**         | 섹션별로 분류된 할일 보드 (아래 섹션 참고)   |
| **담당 일감**         | 나에게 할당된 신규/진행 상태 일감            |
| **보고한 일감**       | 내가 보고한 일감 목록                        |
| **완료된 일감**       | 완료 상태의 일감 목록                        |
| **완료된 개인 작업**  | 완료된 개인 작업 목록                        |

### 일감 항목 구성

- **파란색 점**: 마지막 확인 이후 업데이트된 일감 표시
- **일감 번호 / 제목**
- **우선순위 배지**: 긴급(빨강), 높음(주황), 정상(초록), 낮음(회색)
- **담당자**
- **진행률 바**: 블록 스타일 (시작일~완료예정일 기준)

### 정렬

목록 상단의 정렬 버튼으로 일감을 정렬할 수 있습니다.

| 버튼           | 설명                    |
| -------------- | ----------------------- |
| **등록일**     | 일감 생성일 기준 정렬   |
| **수정일**     | 마지막 수정일 기준 정렬 |
| **완료예정일** | Due date 기준 정렬      |

- 같은 버튼을 다시 클릭하면 오름차순/내림차순 전환
- 다른 버튼 클릭 시 내림차순(최신순)으로 초기화

---

## 4. 새로고침 및 자동 갱신

목록 상단 우측에 새로고침 컨트롤이 있습니다.

| 요소                            | 설명                         |
| ------------------------------- | ---------------------------- |
| **새로고침 버튼** (회전 화살표) | 클릭 시 즉시 일감 목록 갱신  |
| **주기 설정 버튼**              | 자동 갱신 간격 선택 드롭다운 |

### 자동 갱신 주기 옵션

| 옵션 | 설명                                |
| ---- | ----------------------------------- |
| Off  | 자동 갱신 비활성화 (수동 새로고침)  |
| 10s  | 10초마다 자동 갱신                  |
| 30s  | 30초마다 자동 갱신                  |
| 1m   | 1분마다 자동 갱신 (기본값)          |
| 5m   | 5분마다 자동 갱신                   |
| 10m  | 10분마다 자동 갱신                  |

> 설정된 주기는 앱을 재시작해도 유지됩니다.

---

## 5. 일감 상세

목록에서 일감을 클릭하면 상세 화면으로 이동합니다.

### 표시 정보

- 상태, 우선순위, 담당자, 프로젝트
- 진행률, 시작일, 완료예정일
- 설명 (Textile/HTML 마크업 지원)
- 댓글 목록

### 이미지

설명이나 댓글에 포함된 이미지가 자동으로 로드됩니다.

- 이미지 클릭 시 **이미지 뷰어**가 열립니다

### 이미지 뷰어 조작

| 조작                          | 설명                         |
| ----------------------------- | ---------------------------- |
| **마우스 휠**                 | 확대 / 축소                  |
| **마우스 드래그**             | 이미지 이동                  |
| **+ 버튼**                   | 확대                         |
| **- 버튼**                   | 축소                         |
| **초기화 버튼**               | 원본 크기 및 위치로 복원     |
| **X 버튼** 또는 **ESC 키**   | 뷰어 닫기                    |
| **배경 영역 클릭**            | 뷰어 닫기                    |

### 캘린더 등록

상세 화면에서 캘린더 버튼을 클릭하면:

- **Google Calendar**: QR 코드를 스캔하여 일정 등록
- **ICS 파일**: 다운로드하여 캘린더 앱에 가져오기

### 목록으로 돌아가기

좌측 상단 **목록** 버튼을 클릭합니다.

---

## 6. 개인 작업

Redmine과 별개로 로컬에서 관리하는 개인 작업 기능입니다.

### 작업 추가

화면 하단의 **개인 작업 추가** 버튼을 클릭하면 작업 추가 모달이 열립니다.

### 작업 관리

- 개인 작업 항목을 클릭하면 상세 화면으로 이동합니다
- 작업을 완료 처리하거나 삭제할 수 있습니다
- 완료된 작업은 **완료된 개인 작업** 탭에서 확인할 수 있습니다

> 개인 작업 데이터는 앱 내부에 로컬 저장되며 Redmine 서버와는 무관합니다.

---

## 7. 할일 보드

**해야할 일** 탭에서는 섹션별로 작업을 분류하여 관리할 수 있습니다.

### 섹션 관리

- 화면 하단의 **섹션 추가** 버튼으로 새 섹션을 추가할 수 있습니다
- 각 섹션은 고유한 색상으로 구분됩니다
- 섹션 내에서 작업을 분류하여 관리합니다

---

## 8. 하단바

화면 하단에 위치한 바에서 다음 기능을 사용할 수 있습니다.

| 버튼               | 설명                                            |
| ------------------ | ----------------------------------------------- |
| **개인 작업 추가** | 새 개인 작업 추가 모달 열기                     |
| **섹션 추가**      | 할일 보드에 새 섹션 추가 (해야할 일 탭에서만)   |

---

## 9. 타이틀바

커스텀 타이틀바에서 다음 기능을 사용할 수 있습니다.

| 버튼                         | 설명                                     |
| ---------------------------- | ---------------------------------------- |
| **로그아웃** (문 아이콘)     | 로그인 정보 삭제 후 로그인 화면으로 이동 |
| **테마 전환** (달/해 아이콘) | 라이트 모드 / 다크 모드 전환             |
| **항상 위** (핀 아이콘)      | 창을 항상 최상위에 표시 / 해제 토글      |
| **최소화** (- 아이콘)        | 창을 최소화                              |
| **최대화** (확대 아이콘)     | 전체 화면 / 이전 크기 토글               |
| **종료** (X 아이콘)          | 앱 종료                                  |

- 로그인 시 타이틀바에 사용자 ID가 표시됩니다
- 타이틀바 빈 영역을 드래그하여 창을 이동할 수 있습니다
- 타이틀바를 더블 클릭하면 전체 화면 토글됩니다

---

## 10. System Tray

앱은 System tray(작업 표시줄 알림 영역)에 상주합니다.

| 조작                 | 동작                     |
| -------------------- | ------------------------ |
| **좌클릭**           | 창 표시 / 숨기기 토글    |
| **우클릭 > 열기**    | 창 표시                  |
| **우클릭 > 종료**    | 앱 종료                  |

---

## 11. 글로벌 단축키

| 단축키              | 동작                         |
| ------------------- | ---------------------------- |
| `Ctrl + Shift + R`  | 앱 창 표시 / 숨기기 토글     |

> 다른 앱을 사용 중이어도 이 단축키로 즉시 Redmine UI를 호출할 수 있습니다.
> macOS에서는 `Cmd + Shift + R`로 동작합니다.

---

## 12. 테마

라이트 모드와 다크 모드를 지원합니다.

- 최초 실행 시 OS 설정에 따라 자동 선택
- 타이틀바의 테마 버튼으로 전환
- 선택한 테마는 앱 재시작 후에도 유지

---

## 13. 주의사항

- **Always on Top**: 앱 창은 기본적으로 항상 최상위에 표시됩니다. 타이틀바의 핀 버튼으로 해제/고정할 수 있습니다.
- **닫기 동작**: X 버튼을 누르면 앱이 종료됩니다. 창만 숨기려면 System tray 좌클릭 또는 `Ctrl+Shift+R`을 사용하세요.
- **네트워크**: Redmine 서버에 접근 가능한 네트워크 환경이 필요합니다.
